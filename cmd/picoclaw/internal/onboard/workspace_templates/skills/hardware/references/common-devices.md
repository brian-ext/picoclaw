# Common Devices

This file contains quick notes for common sensor/display devices used with I2C/SPI.

## AHT20

- Typical address: `0x38`

## BME280

- Common addresses: `0x76`, `0x77`

## SSD1306 OLED

- Common address: `0x3C`

## MPU6050

- Common address: `0x68`

## DS3231

- Common address: `0x68`

## INA219

- Common address: `0x40`

## PCA9685

- Common address: `0x40`
